# Dev Layout

This repo contains 3 separate parts:

1) Heidi backend (Python) – repo root
2) Heidi UI (custom) – ./ui
3) Open WebUI (separate project to theme/polish) – ./openwebui

Rules:
- If task is Open WebUI polish, ONLY edit files under ./openwebui
- Heidi UI lives under ./ui
- Backend is started from repo root
